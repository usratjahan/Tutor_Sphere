# Tutor_Sphere
TutorSphere is a peer learning platform designed to help students find trustworthy academic support from verified mentors within their institution. In modern education, learners often struggle to find dependable peers who can provide tutoring support in specific subjects. This system solves that problem by enabling two essential roles; mentors (peer tutors) and learners (students seeking help) to connect digitally through a simple matchmaking process.
 Features in Detail
1 Admin Features
Approve or reject mentor verification requests
Manage user accounts
View system statistics
Monitor tutor-student interactions
Remove abusive or fake accounts
Maintain system consistency

2 Tutor Features
Create and update academic profile
Search for students
Receive and accept/reject student tutoring requests
Upload verification documents
Receive notifications
Maintain personal dashboard

3 Student Features
Create academic profile
Search for verified tutors by subject
Send tutoring requests
Track request status
Maintain personal dashboard

4 How Admin Uses Dashboard
Logs into the system
Views pending tutor verification requests
Reviews uploaded documents
Approves or rejects mentor
Manages user accounts
Monitors requests and platform data

5 How Tutor & Student Use the Platform
Tutor:
Logs in
Completes profile
Uploads verification document
Searches for students
Accepts/rejects tutoring requests
Views notifications
Student:
Logs in
Creates/updates profile
Searches for suitable tutors
Sends request
Waits for tutor response
